# Responsible Use

`LottoMind` 是一个统计分析与界面实验项目，不是彩票销售、投注建议或收益承诺工具。

## 使用边界

- 推荐号码仅用于历史数据分析和产品演示
- 不提供“必中”“保底”“套利”等承诺
- 不替代用户对法律、平台规则和资金风险的判断

## 合规提醒

- 请遵守你所在地区关于彩票、数据抓取和 AI 服务使用的法律法规
- 不要将本项目包装为官方彩票应用或授权服务
- 不要在营销或截图中使用误导性收益表述

## 数据与账号

- 你需要自行申请并管理 AI 服务商 API Key
- 请勿把真实 Key、账本截图或个人信息提交到公开仓库
- 当你主动执行联网请求时，Key 会直接发送到相应服务商完成鉴权

## 项目定位

- 开源实验
- 本地优先
- 可审计
- 不托管用户隐私数据
