# Responsible Use

`LottoMind` is a statistics and interface experiment. It is not a ticket-selling product, financial advice tool, or guaranteed-outcome system.

## Usage Boundaries

- Recommendations are for historical analysis and product demonstration only
- The project does not promise guaranteed wins, hedging, or arbitrage outcomes
- It does not replace user judgment about law, platform rules, or financial risk

## Compliance Notes

- Follow the laws and platform policies that apply in your region
- Do not present this project as an official lottery app or authorized service
- Do not use misleading profit claims in screenshots, marketing, or derivative projects

## Data And Accounts

- You must manage your own AI-provider API key
- Do not commit real keys, ledger screenshots, or personal data to a public repository
- When you trigger online requests, the key is sent directly to the selected provider for authentication

## Project Positioning

- Open-source experiment
- Local-first
- Auditable
- No hosted user-private backend
