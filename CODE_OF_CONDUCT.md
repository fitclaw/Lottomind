# Code Of Conduct

为了让 `LottoMind` 社区保持开放、友善、可协作，请所有参与者遵守以下约定。
To keep the `LottoMind` community open, kind, and collaborative, please follow these expectations.

- 尊重不同背景、经验和观点，避免人身攻击、羞辱或骚扰行为。
- 讨论聚焦代码、产品和事实，不对个人动机做恶意揣测。
- 不发布他人的隐私信息、密钥、账号截图或其他敏感材料。
- 对新手保持耐心，欢迎提问、复现、补文档和改进测试。
- 如果发现功能涉及合规、隐私或安全风险，优先提醒并协助修正。
- 维护者有权删除不当内容，并在必要时限制反复违规的参与者。
- Respect different backgrounds, experience levels, and viewpoints; avoid harassment, insults, or attacks.
- Keep discussion focused on code, product behavior, and facts rather than personal motives.
- Do not post private information, secrets, account screenshots, or other sensitive material.
- Be patient with newcomers and welcome questions, reproductions, docs fixes, and test improvements.
- If something raises compliance, privacy, or security concerns, surface it early and help correct it.
- Maintainers may remove inappropriate content and restrict repeated abuse when necessary.

如果你在社区互动中感到不适，可以通过仓库维护渠道提出反馈；提交反馈时请尽量附上最小必要上下文，不要包含额外隐私信息。
If an interaction makes you uncomfortable, please report it through the repository maintenance channel with the minimum necessary context and without extra private data.
