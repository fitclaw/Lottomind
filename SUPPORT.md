# Support

如果你想为 `LottoMind` 提问、报错或提建议，推荐按下面的入口使用：

- 使用问题：提交 `question` 类型 Issue
- 缺陷反馈：提交 `bug report`
- 功能建议：提交 `feature request`
- 安全问题：参考 [SECURITY.md](/Users/vtx/Dev/LettoMind/SECURITY.md)

提交前请注意：

- 不要粘贴真实 API Key、Cookie、Token 或个人隐私数据
- 不要上传包含账本、付款或身份信息的截图
- 如果问题与 AI 服务商接口有关，请说明服务商、模型和错误现象

如果你准备贡献代码，请继续阅读 [CONTRIBUTING.md](/Users/vtx/Dev/LettoMind/CONTRIBUTING.md)。
