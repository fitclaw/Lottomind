# Support

如果你想为 `LottoMind` 提问、报错或提建议，推荐按下面的入口使用。
If you need help, want to report a bug, or want to suggest an improvement, start with one of the routes below.

- 使用问题：提交 `question` 类型 Issue
- 缺陷反馈：提交 `bug report`
- 功能建议：提交 `feature request`
- 安全问题：参考 [SECURITY.md](./SECURITY.md)
- Usage questions: open a `question` issue
- Bug reports: use the `bug report` template
- Feature ideas: use the `feature request` template
- Security concerns: read [SECURITY.md](./SECURITY.md)

## 提交前请注意 / Before You Submit

- 不要粘贴真实 API Key、Cookie、Token 或个人隐私数据
- 不要上传包含账本、付款或身份信息的截图
- 如果问题与 AI 服务商接口有关，请说明服务商、模型和错误现象
- Do not paste real API keys, cookies, tokens, or personal data
- Do not upload screenshots containing ledger data, payment data, or identity information
- If the problem involves an AI provider, mention the provider, model, and observed behavior

如果你准备贡献代码，请继续阅读 [CONTRIBUTING.md](./CONTRIBUTING.md)。
If you plan to contribute code, continue with [CONTRIBUTING.md](./CONTRIBUTING.md).
